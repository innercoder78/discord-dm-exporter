Discord DM Log Exporter

Discord Web is not open.

This extension only works on https://discord.com in Chrome.
Open Discord Web to continue.

Compatibility:

This extension only works with Discord Web in Chrome.
It does not work with the Discord desktop app, mobile app, or tablet app.

Scope:

This extension is designed only for one-on-one DMs.
It does not support server channels, group chats, threads, forums, or voice channels.

For best results, enter both Discord display names without server tags. If these are blank, speaker detection may be less accurate.
Date Range mode requires both a start date and an end date.

Choose both dates, or check EVERYTHING.

Collects a privacy-safe debug log for troubleshooting.
